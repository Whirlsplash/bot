## Usage
1. Clone via `git clone git@github.com:Whirlsplash/bot.git`
2. Set `DISCORD_TOKEN` environment variable in `.env` file, e.g., [`.env.example`](./.env.example)
3. Install dependencies and build via `cargo build`
4. Run via `cargo run`
